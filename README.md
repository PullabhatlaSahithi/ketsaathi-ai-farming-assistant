# 🌾 Ketsaathi - AI Smart Farming Assistant

Ketsaathi is a responsive web application designed to help farmers with step-by-step guidance from soil preparation to market selling.

## 🚀 Features
- 🌐 Multi-language support (Telugu, Hindi, English)
- 🤖 AI-based farming assistant interface
- 📊 Step-by-step farming guidance
- 💬 Chatbot interaction
- 📱 Fully responsive design

## 🛠️ Tech Stack
